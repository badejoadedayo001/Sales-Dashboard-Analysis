 PROJECT OVERVIEW
This project analyzes sales, customer, and inventory data to identify revenue trends, customer behavior, product performance, major age group, and stock levels.

 TOOLS USED
i. Excel – Data cleaning and initial analysis
ii. Power Query – data transformation and merging
iii. Power BI – Dashboard creation and visualization
iv. DAX – Creating calculated measures 

 BUSINESS QUESTIONS ANSWERED
i. Which products generate the most revenue?
ii. Which customer type contributes the most sales?
iii. Which location performs best?
iv. Which age group generates the highest revenue?
v. Which products have low inventory?

 KEY INSIGHTS
i. Samsung S25 generated the highest revenue.
ii. Corporate customers generated the highest revenue.
iii. Kano had the strongest sales performance.
iv. Age 18-25 generated the highest revenue. 
v. Airpods pro had the lowest stock level. 


 RECOMMENDATIONS
i. Increase inventory monitoring for low stock products.
ii. Focus marketing efforts on high-value customer groups.
iii. Maintain stock availability for top selling product.
